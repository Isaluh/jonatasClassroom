Esta aplicação é um Sistema de Cadastro de Alunos desenvolvido com Angular. O projeto tem como objetivo fornecer um formulário simples para o cadastro de alunos, tudo isso utilizando rotas para navegação entre as páginas.

Estrutura:

  1. HeaderComponent
  2. MainComponent
  3. FormularioView
  4. InformacaoView


Navegação:

  • Rota (/) leva a View de formulário;
  • Rota (/info) leva a View de informações do projeto.


A aplicação utiliza o two-way data binding com [(ngModel)] para capturar as informações dos alunos no formulário e exibi-lo em tempo real. A lista de alunos é exibida com a diretiva ngFor e, caso não haja alunos cadastrados, uma mensagem é mostrada, utilizando a diretiva ngIf.

A navegação entre as páginas é simplificada por meio de links no cabeçalho, usando a diretiva routerLink.

Esse projeto é uma forma simples e prática de aprender conceitos importantes do Angular, como componentes, rotas e data binding.
